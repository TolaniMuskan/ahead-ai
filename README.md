# 🚀 AHEAD – AI Productivity Companion

> An AI-powered productivity companion that proactively helps users plan, prioritize, and complete tasks before deadlines are missed.

Built for the **Vibe2Ship Hackathon** using **Google Gemini API** and **Firebase**.

---

## 📌 Problem Statement

Build an AI-powered productivity companion that proactively assists users in planning, prioritizing, and completing tasks before deadlines are missed.

Unlike traditional reminder apps, the solution should help users take meaningful action through intelligent planning and personalized recommendations.

---

# 💡 Solution

AHEAD is an AI-powered productivity assistant that goes beyond reminders.

Users simply enter a goal or upcoming deadline, and AHEAD uses Google's Gemini API to:

- Analyze the task
- Estimate success probability
- Recommend the best strategy
- Simulate future outcomes
- Generate a personalized execution plan
- Suggest recovery strategies if things go off track

The application helps users make smarter decisions before deadlines are missed rather than simply reminding them.

# 🖼 Application Flow

1. Login using Google
2. Enter a goal or deadline
3. Gemini analyzes the request
4. AI generates:
   - Personalized strategy
   - Success prediction
   - Future simulations
   - Execution timeline
   - Recovery plan
5. User follows the recommended action plan

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/TolaniMuskan/ahead-ai.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add your Firebase and Gemini API credentials.

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```
